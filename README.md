## ⚙️ How to Run

```bash
# Clone the repo
git clone <your-repo-url>

# Navigate to project
cd auth-system-fastapi/backend

# Create virtual environment
python -m venv venv

# Activate (Windows)
venv\Scripts\activate

# Install dependencies
pip install fastapi uvicorn

# Run server
uvicorn main:app --reload
```
